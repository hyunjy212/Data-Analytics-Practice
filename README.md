# 현지윤의 데이터 분석 독학 기록

## 📅 예정된 폴더 구조

- `week02_pandas/` : Pandas를 활용한 데이터 다루기
- `week03_sql/` : SQL 기초 및 쿼리 실습
- `mini_projects/` : 개인 분석 프로젝트 저장
